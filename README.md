# Animal-Crossing-Critter-Tracker
A rough website that can be used to find info on the critters in the Animal Crossing games, and allow the user to save and search by catch and donation status. A test project for a class to familiarize with APIs and React.

This project was only ever made for personal use and required a personal API key to the Nookipedia API which I will not be sharing, but can easily be obtained following the instructions on the Nookipedia Website.

Running this will require the user to create their own SQL database, but the tables should be easily made and populated by running the `migrate.js` file under the server directory.

The backend of the website is ran on an Express JS server. The front end I run through a Vite server as I am still developing. More details on setting up and running the projects can be found in the `README.md` in the next directory.
